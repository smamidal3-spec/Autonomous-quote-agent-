# Algorithms Used

## Risk Profiling
- Random Forest classifier + strict heuristic override
- SHAP feature attribution for top explanation

## Fraud Detection
- Isolation Forest anomaly score
- Gradient boosting fraud probability
- Rule engine for deterministic escalation flags

## Conversion Prediction
- XGBoost probability estimation
- Banding into LOW/MEDIUM/HIGH conversion classes

## Personalization
- K-Means customer segmentation
- Segment-to-plan mapping and dynamic add-on enrichment

## Decision Routing
- Rule-based state machine combining risk, conversion, premium, and fraud states
- Confidence score and escalation reason generation
